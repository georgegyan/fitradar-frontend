## FitRadar
FitRadar is a location-based web platform that helps users discover and book nearby gyms in real-time.

## Features (MVP)
Find nearby gyms based on your current location
View gym details
Book gym sessions
Gym owners can submit their gyms

## Tech Stack
Backend: Django + Django REST Framework
Frontend: React / Next.js (planned)
Database: SQLite (for development)

## Project Status
Currently in development (MVP phase)
